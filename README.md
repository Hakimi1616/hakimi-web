## 🌟 Premium Features

### 1. Dynamic Inventory Engine
* **Real-time Rendering:** Products are injected into the DOM dynamically using high-performance JavaScript arrays.
* **Stock Management Logic:** The system tracks inventory levels. If a product reaches zero stock, it automatically displays a "Depleted" badge and disables the purchase functionality.

### 2. AI-Driven Smart Filtering
* **Live Search:** An instant search bar that filters products by name or category as the user types.
* **Dynamic Price Range:** A specialized price filter allows users to set a maximum budget, updating the marketplace view in real-time.

### 3. Advanced UI/UX (Glassmorphism)
* **Modern Aesthetics:** Built with a custom Tailwind CSS configuration using the "Space Grotesk" typography for a futuristic look.
* **Responsive Architecture:** Fully optimized for Mobile, Tablet, and Desktop displays.
* **Product Detail Modal:** A premium pop-up interface that provides deep-dive specifications for each product without reloading the page.

### 4. Interactive Transaction Workflow
* **Sliding Cart System:** A sleek, side-bar shopping bag that uses CSS transitions for a smooth user flow.
* **Automated Calculations:** Real-time subtotal and tax calculation with professional currency formatting.

## 📖 Installation & Usage
1.  Clone the repository or download the `ziyach_ultimate_store.html` file.
2.  Open the file in any modern web browser (Chrome, Firefox, or Edge).
3.  **To Add Items:** Click the "+" icon or the "Add to Collection" button.
4.  **To Filter:** Use the search bar or the price slider to narrow down your tech selection.
5.  **To Checkout:** Open the bag and click "Proceed to Checkout" to trigger the secure gateway simulation.
